ภาคผนวกใช้เก็บข้อมูลอ้างอิงที่ผู้อ่านจะกลับมาเปิดดูบ่อย เช่น HTTP status code, ปัญหา EF Core, ปัญหา JWT และโครงสร้าง final project

## รายการภาคผนวก

- โครงสร้าง Final Project
- Minimal API เทียบกับ Controller
- ปัญหาที่พบบ่อยใน EF Core
- ปัญหาที่พบบ่อยใน JWT
- ตาราง HTTP Status Code ที่ใช้บ่อย
- ตัวอย่าง REST Client และ Postman Collection
