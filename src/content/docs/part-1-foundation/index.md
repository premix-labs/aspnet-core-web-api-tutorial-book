ภาคนี้ทำให้ผู้อ่านเข้าใจภาพรวมก่อนลงมือเขียน API จริง จุดสำคัญคือรู้ว่า ASP.NET Core Web API ทำงานอยู่ตรงไหนในระบบ backend และเครื่องมือแต่ละตัวมีหน้าที่อะไร

## วิธีเรียนภาคนี้

ให้อ่านตามลำดับและทำ checkpoint ท้ายบทให้ผ่านก่อนข้ามไปบทถัดไป โดยเฉพาะบทติดตั้งเครื่องมือและบทสร้างโปรเจกต์แรก

หนังสือใช้ Visual Studio Code, PowerShell และ `dotnet` CLI เป็นเส้นทางหลัก แต่ถ้าคุณใช้ Visual Studio ก็เรียนตามได้ โดยยังควรตรวจคำสั่งผ่าน terminal ให้ได้เหมือนกัน

ภาคนี้ยังไม่เน้น business logic หนัก เป้าหมายคือทำให้คุณเข้าใจคำพื้นฐานและรันโปรเจกต์แรกได้อย่างมั่นใจ ก่อนเข้าสู่ Controller และ CRUD ในภาคถัดไป

## บทในภาคนี้

- บทที่ 1: ASP.NET Core Web API คืออะไร
- บทที่ 2: ติดตั้งเครื่องมือที่จำเป็น
- บทที่ 3: สร้างโปรเจกต์แรก
- บทที่ 4: เข้าใจโครงสร้างไฟล์ของ ASP.NET Core
- บทที่ 5: พื้นฐาน HTTP, REST และ JSON

## สิ่งที่ต้องได้หลังจบภาคนี้

- อธิบายได้ว่า Web API คืออะไร
- สร้างและรัน ASP.NET Core project ได้
- รู้จักไฟล์สำคัญ เช่น `Program.cs`, `appsettings.json` และ `*.csproj`
- เข้าใจ request, response, method, route, JSON และ status code เบื้องต้น
