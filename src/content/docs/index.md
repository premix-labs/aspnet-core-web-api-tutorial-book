หนังสือเล่มนี้สอนสร้าง Web API ด้วย ASP.NET Core แบบค่อยเป็นค่อยไป โดยเริ่มจากพื้นฐาน HTTP, REST, Controller, DTO, Service, Repository, Entity Framework Core, Authentication, Authorization, Testing, Docker และแนวทางเตรียมโปรเจกต์ก่อน deploy

แนวทางของหนังสือคือให้ผู้อ่านสร้างโปรเจกต์เดียวต่อเนื่องตั้งแต่ต้นจนจบ เพื่อให้เห็นภาพการทำงานจริงของระบบ backend มากกว่าการเรียนแยกหัวข้อแบบกระจัดกระจาย

## โปรเจกต์หลักของหนังสือ

โปรเจกต์หลักคือ **Secure Admin Web API** ซึ่งประกอบด้วยระบบสมัครสมาชิก, เข้าสู่ระบบ, JWT, role-based authorization, CRUD ผู้ใช้, pagination, validation, error handling, logging, automated tests และ Docker Compose

## Stack ที่ใช้

- .NET 10 LTS
- ASP.NET Core Web API แบบ Controller
- Entity Framework Core
- SQL Server
- JWT Bearer Authentication
- OpenAPI
- xUnit
- Docker และ Docker Compose
- Astro และ Starlight สำหรับทำเว็บไซต์หนังสือ

## เหมาะกับใคร

- คนที่เริ่มเรียน ASP.NET Core Web API
- คนที่เคยเขียน C# พื้นฐานแล้วอยากต่อยอดเป็น backend
- คนที่ต้องการเข้าใจโครงสร้างโปรเจกต์ที่ใช้ทำงานจริง
- คนที่อยากมีโปรเจกต์ตัวอย่างไว้ใส่ portfolio หรือใช้เป็น template เริ่มงาน

## วิธีอ่าน
