ASP.NET Core รองรับทั้ง Minimal API และ Controller ทั้งสองแบบใช้สร้าง Web API ได้จริง

## Minimal API

เหมาะกับ endpoint จำนวนน้อย service ขนาดเล็ก หรือ prototype ที่ต้องการ code สั้น

## Controller

เหมาะกับระบบที่มีหลาย endpoint ต้องใช้ attribute, validation, authorization, versioning หรือแยกโครงสร้างเป็นทีม

หนังสือนี้ใช้ Controller เพราะเหมาะกับการสอนมือใหม่ให้เห็นโครงสร้าง backend แบบเป็นระบบ
